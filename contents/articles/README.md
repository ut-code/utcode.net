# 記事ページ

## frontmatter

| キー          | 必須 | 説明                                                                           |
| ------------- | ---- | ---|
| `title`       | ✅   | 記事タイトル|
| `date`        | ✅   | **記事を書いた** 日。|
| `image`       | ✅   | イメージ画像 |
| `fit` | | (画像) object-fit。 default = "cover" |
| `position` | | (画像) object-position。 (縦軸上から 横軸左から)。default = "50% 50%" |
| `categories`  |      | 何なんだろうね。 |
| `author`|  | 著者の id |
